📖 第一章：Introduction (簡介)

網路概論基礎介紹 — 網路的定義、類型與拓撲

網路的定義

網路（Network）是將多台電腦與設備透過傳輸媒介連接起來，使彼此能夠交換資料與共享資源的系統。

網路三要素：至少兩台設備、傳輸媒介（纜線或無線）、通訊協定（共同語言規則）。

網路的分類 (依規模)

類型 全名 涵蓋範圍
PANPersonal Area Network個人範圍（藍芽、USB）
LANLocal Area Network建築物內（校園、辦公室）
MANMetropolitan Area Network城市範圍
WANWide Area Network國家/全球（網際網路）

網路拓撲 (Topology)

拓撲 說明 優點 缺點
匯流排 (Bus) 所有設備共享一條主幹線 佈線簡單、成本低 主幹故障全網中斷、碰撞問題
星狀 (Star) 所有設備連接到中央節點 易管理、單點故障不影響全網 中央節點故障全網中斷
環狀 (Ring) 設備首尾相連成環 傳輸穩定、無碰撞 單點故障影響全網
網狀 (Mesh) 每個設備互相連接 高冗餘、高可靠性 佈線成本高
樹狀 (Tree) 星狀的層級延伸 易擴充 根節點故障影響大

傳輸模式

  • 單工 (Simplex)：單向傳輸，如廣播、電視
  • 半雙工 (Half-Duplex)：雙向但不能同時，如對講機
  • 全雙工 (Full-Duplex)：雙向可同時，如電話、網路交換器

📡 第二章：Communication (通訊)

網路通訊基礎概念 — 訊號、傳輸媒介與多工技術

資料通訊系統五大元件

  1. 訊息 (Message)：要傳送的資料（文字、圖像、聲音）
  2. 傳送端 (Sender)：發出訊息的設備
  3. 接收端 (Receiver)：接收訊息的設備
  4. 傳輸媒介 (Medium)：訊號傳遞的通道
  5. 協定 (Protocol)：通訊規則的集合

類比 vs 數位訊號

特性 類比訊號 (Analog) 數位訊號 (Digital)
波形連續平滑波形離散階梯狀（0/1）
衰減會衰減，需放大器會衰減，需中繼器
雜訊放大時同時放大雜訊可還原，抗干擾性強
頻寬使用較高較低

傳輸媒介

有線傳輸
  • 雙絞線 (UTP/STP)：成本低、易安裝，常用於乙太網路
  • 同軸纜線 (Coaxial)：抗干擾佳，早期用於有線電視
  • 光纖 (Fiber Optic)：高頻寬、長距離、抗電磁干擾
無線傳輸
  • 無線電波：Wi-Fi、行動通訊
  • 微波：衛星通訊、點對點
  • 紅外線：短距離遙控

多工技術 (Multiplexing)

  • FDM (頻分工)：不同訊號使用不同頻率，如廣播電台
  • TDM (時分工)：不同訊號輪流使用時間槽，如T1/E1線路
  • WDM (波分工)：光纖中使用不同波長的光
  • CDMA (碼分工)：使用不同編碼同時傳輸，如3G

🔧 第三章：Equipment (設備)

網路設備 — 從實體層到網路層的設備功能與應用

網路設備層級分類

設備 OSI層級 功能 定址方式
中繼器 (Repeater)Layer 1訊號放大延伸距離無
集線器 (Hub)Layer 1多埠中繼器，廣播到所有埠無
橋接器 (Bridge)Layer 2分割碰撞域，過濾幀MAC位址
交換器 (Switch)Layer 2多埠橋接器，MAC表轉送MAC位址
路由器 (Router)Layer 3路由選擇，分割廣播域IP位址
閘道器 (Gateway)Layer 4-7協定轉換，連接不同網路依協定而定

碰撞域 vs 廣播域：Hub不分割碰撞域，Switch每個埠獨立碰撞域，Router每個埠獨立廣播域。

交換器 (Switch) 進階功能

  • MAC位址學習：記錄來源MAC與對應埠號
  • Store-and-Forward：完整接收幀後檢查錯誤再轉送
  • Cut-Through：讀取目的MAC後立即轉送，低延遲
  • VLAN：將單一交換器分割為多個邏輯網路
  • STP (生成樹協定)：防止網路迴路
  • Link Aggregation：多條鏈路合併增加頻寬

路由器 (Router)

  • 路由表：記錄目的地網路與下一跳位址
  • 靜態路由：管理員手動設定
  • 動態路由：透過路由協定自動學習（RIP、OSPF、BGP）
  • NAT：私有IP轉換為公有IP
  • ACL：存取控制清單，過濾封包

📜 第四章：Protocol (協定)

網路通訊協定 — OSI模型、TCP/IP與重要協定

通訊協定三要素

  • 語法 (Syntax)：資料格式與結構
  • 語意 (Semantics)：欄位意義與控制資訊
  • 時序 (Timing)：傳送時機與速度匹配

OSI 七層模型

由ISO制定的標準網路架構模型：

層級名稱功能協定/技術
L7應用層使用者應用服務HTTP, FTP, SMTP, DNS
L6表達層資料格式轉換、加密SSL/TLS, JPEG
L5會議層會話管理NetBIOS, RPC
L4傳輸層端對端傳輸、流量控制TCP, UDP
L3網路層路由、邏輯定址IP, ICMP, ARP
L2資料連結層幀傳輸、MAC定址Ethernet, PPP
L1實體層位元傳輸、電氣訊號RS-232, RJ-45

口訣：由上到下 — 應表會傳網資實

TCP/IP 模型

網際網路實際使用的四層架構：

TCP/IP層對應OSI主要協定
應用層L5+L6+L7HTTP, FTP, SMTP, DNS, SSH
傳輸層L4TCP（可靠）、UDP（不可靠）
網際網路層L3IP, ICMP, ARP
網路存取層L1+L2Ethernet, Wi-Fi, PPP

TCP vs UDP

TCP
  • 面向連線（三次握手）
  • 可靠傳輸、流量控制
  • 擁塞控制（慢啟動）
  • 資料按序送達
  • 適用：HTTP, FTP, Email
UDP
  • 無連線
  • 不可靠、盡力傳送
  • 低延遲、高效率
  • 無流量/擁塞控制
  • 適用：DNS, VoIP, 串流

重要協定彙整

協定全名功能埠號
HTTPHypertext Transfer Protocol網頁傳輸80
HTTPSHTTP Secure加密網頁傳輸443
FTPFile Transfer Protocol檔案傳輸21
SMTPSimple Mail Transfer Protocol郵件傳送25
DNSDomain Name System域名解析53
DHCPDynamic Host Configuration Protocol自動配發IP67/68
ARPAddress Resolution ProtocolIP→MAC轉換N/A
ICMPInternet Control Message Protocol錯誤報告（ping）N/A

🔌 第五章：Ethernet (乙太網路)

乙太網路技術 — IEEE 802.3 標準系列

乙太網路演進

標準速度傳輸媒介最大距離
10BASE510 Mbps粗同軸纜線500m
10BASE210 Mbps細同軸纜線185m
10BASE-T10 MbpsUTP Cat3100m
100BASE-TX100 MbpsUTP Cat5100m
1000BASE-T1 GbpsUTP Cat5e100m
10GBASE-T10 GbpsUTP Cat6a100m

命名規則：速度 + BASE + 媒介代碼。
例如：1000BASE-T = 1000 Mbps、基頻(Baseband)、雙絞線(Twisted pair)

CSMA/CD 存取控制

傳統半雙工乙太網路使用 CSMA/CD 控制媒體存取：

  1. 載波偵聽：傳送前先偵聽線路是否空閒
  2. 傳送：空閒則開始傳送資料
  3. 碰撞偵測：傳送時持續監聽是否發生碰撞
  4. Jam訊號：碰撞發生時傳送擁塞訊號通知所有設備
  5. 指數退避：隨機等待後重新嘗試傳送

注意：使用交換器的全雙工模式不需要CSMA/CD，因為發送和接收使用不同線對，不會碰撞。

乙太網路幀格式

| Preamble | SFD | Dest MAC | Src MAC | Type | Payload | FCS | | 7B | 1B | 6B | 6B | 2B |46-1500B | 4B |
  • 前導碼 (Preamble)：7 bytes，同步訊號
  • SFD：1 byte，幀起始分隔符
  • 目的/來源MAC：各6 bytes
  • Type/Length：2 bytes，上層協定類型
  • Payload：46-1500 bytes（不足需填充）
  • FCS：4 bytes，CRC-32錯誤檢查

最小幀大小：64 bytes（確保CSMA/CD正確偵測碰撞）
最大幀大小：1518 bytes（MTU=1500）

MAC 位址

  • 48位元（6 bytes），全球唯一硬體位址
  • 前24位元：OUI（IEEE分配給製造商）
  • 後24位元：製造商分配
  • 廣播位址：FF:FF:FF:FF:FF:FF

🌐 第七章：LAN (區域網路)

區域網路技術 — IEEE 802標準、VLAN、無線LAN

LAN 定義與特性

區域網路（Local Area Network）是在有限地理範圍內連接多台設備的通訊網路。

  • 範圍：單一建築或校園
  • 速度：10 Mbps ~ 100 Gbps
  • 所有權：單一組織擁有管理
  • 錯誤率：低（傳輸品質高）

IEEE 802 標準家族

標準說明
802.1橋接、VLAN、STP
802.3乙太網路 (CSMA/CD)
802.11無線區域網路 (Wi-Fi)
802.15無線個人網路 (Bluetooth)
802.16寬頻無線 (WiMAX)

VLAN 虛擬區域網路

VLAN在單一實體交換器上建立多個邏輯網路：

  • 優點：安全性、減少廣播風暴、管理彈性
  • Port-based：依據交換器埠號分配VLAN
  • IEEE 802.1Q：在乙太網路幀插入4 bytes VLAN標籤
  • VLAN ID：12 bits，範圍1-4094
  • Trunk：跨交換器傳送多個VLAN流量

802.1Q標籤結構：TPID(2B, 0x8100) + Priority(3b) + CFI(1b) + VLAN ID(12b)

無線區域網路 (WLAN)

WLAN基於IEEE 802.11標準，使用無線電波傳輸：

標準頻段最大速率特色
802.11b2.4 GHz11 Mbps最早普及
802.11g2.4 GHz54 MbpsOFDM調變
802.11a5 GHz54 Mbps干擾較少
802.11n (Wi-Fi 4)2.4/5 GHz600 MbpsMIMO技術
802.11ac (Wi-Fi 5)5 GHz6.9 GbpsMU-MIMO
802.11ax (Wi-Fi 6)2.4/5/6 GHz9.6 GbpsOFDMA

🌍 第八章：IP (網際網路協定)

網際網路協定 — IPv4/IPv6定址、子網路切割、NAT、路由協定

IPv4 定址

IPv4 位址為32位元，以點分十進制表示：

Class範圍預設遮罩用途
A1.0.0.0 - 126.255.255.255/8大型網路
B128.0.0.0 - 191.255.255.255/16中型網路
C192.0.0.0 - 223.255.255.255/24小型網路
D224.0.0.0 - 239.255.255.255N/A多播
E240.0.0.0 - 255.255.255.255N/A實驗用

私有IP位址 (RFC 1918)

  • 10.0.0.0/8：10.0.0.0 - 10.255.255.255
  • 172.16.0.0/12：172.16.0.0 - 172.31.255.255
  • 192.168.0.0/16：192.168.0.0 - 192.168.255.255
  • 127.0.0.0/8：Loopback（本機回送）
  • 169.254.0.0/16：APIPA（自動私有IP）

子網路切割 (Subnetting)

將一個大網路劃分為多個小網路，提高位址利用率：

  • 子網路數 = 2^(借用位元數)
  • 可用主機數 = 2^(剩餘主機位元數) - 2
  • 減2原因：網路位址（全0）和廣播位址（全1）

CIDR表示法：IP位址/前綴長度
例：192.168.1.0/24 表示前24位元為網路部分，後8位元為主機部分

IPv4 封包結構

IPv4標頭最小20 bytes，主要欄位：

  • Version(4b)：IP版本
  • IHL(4b)：標頭長度
  • Total Length(16b)：封包總長
  • TTL(8b)：存活時間，每跳減1
  • Protocol(8b)：上層協定（TCP=6, UDP=17）
  • Source/Dest IP(各32b)

NAT 網路位址轉譯

  • 靜態NAT：一對一映射
  • 動態NAT：從公有IP池取用
  • PAT (NAPT)：多對一，使用埠號區分（家用路由器最常用）

路由協定

協定類型度量適用範圍
RIP距離向量跳數小型網路
OSPF鏈路狀態成本中大型網路
EIGRP混合型複合度量思科網路
BGP路徑向量AS路徑網際網路骨幹

IPv6 簡介

  • 128位元位址空間，解決IPv4枯竭
  • 冒號十六進制表示：2001:db8::1
  • 內建IPSec安全性
  • SLAAC無狀態自動設定IP
  • 無廣播，使用多播取代
  • 無NAT需求（位址充足）
